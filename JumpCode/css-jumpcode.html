🎯 CSS Selectors

Universal & Element Selectors
Target all or specific HTML elements
* { } /* All elements */ div { } /* All divs */ .class { } /* All elements with class */ #id { } /* Element with ID */ div.class { } /* Divs with class */ div#id { } /* Div with specific ID */
Combinator Selectors
Select based on relationships
div p { } /* All p inside div (descendants) */ div > p { } /* Direct p children of div */ div + p { } /* p immediately after div */ div ~ p { } /* All p that follow div */
Pseudo-classes & Pseudo-elements
Special element states and parts
a:hover { } /* Mouse over link */ a:active { } /* Active link */ a:visited { } /* Visited link */ input:focus { } /* Focused input */ li:first-child { } /* First child */ li:last-child { } /* Last child */ li:nth-child(2n) { } /* Every 2nd child */ p::first-line { } /* First line of paragraph */ p::before { } /* Before content */ p::after { } /* After content */
Attribute Selectors
Select by attributes
input[type="text"] { } /* Input with type=text */ a[href^="https"] { } /* Link starting with https */ a[href$=".pdf"] { } /* Link ending with .pdf */ a[href*="example"] { } /* Link containing example */ img[alt] { } /* Images with alt attribute */

📱 Responsive Design

Media Queries
Styles for different screen sizes

🚀 Advanced Features

CSS Variables (Custom Properties)
Reusable values throughout CSS

Clip-path
Create complex shapes
clip-path: circle(50%); clip-path: polygon(50% 0%, 100% 50%, 50% 100%, 0% 50%); clip-path: inset(10px 20px 30px 40px); clip-path: ellipse(50% 75%);
Backdrop Filter
Blur background behind element
backdrop-filter: blur(10px); backdrop-filter: brightness(80%); background-color: rgba(255, 255, 255, 0.7); /* Combine for effect */
Overflow
Handle content overflow
overflow: visible; /* Default - show overflow */ overflow: hidden; /* Hide overflow */ overflow: scroll; /* Always show scrollbar */ overflow: auto; /* Show scrollbar if needed */ overflow-x: auto; /* Horizontal scroll only */ overflow-y: hidden; /* Hide vertical overflow */
💡 Pro Tips:
  • Use rem for font sizes and em for component-relative sizes
  • Always include box-sizing: border-box in your reset CSS
  • Use CSS variables for colors, spacing, and other repeated values
  • Mobile-first approach: style for mobile, then add media queries for larger screens
  • Combine flexbox for 1D layouts and grid for 2D layouts
  • Use transform and opacity for animations (better performance)
  • Always test on different devices and browsers
  • Keep specificity low by using classes instead of IDs
  • Use CSS preprocessors (Sass, Less) for complex projects
